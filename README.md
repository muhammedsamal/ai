# ai

A CLI tool for AI-powered prompts.

## Installation

```bash
go get github.com/muhammedsamal/ai
```

## Setup

```bash
cp .env.example .env
```

## Build

```bash
go build -o ai && sudo cp ai /usr/local/bin/
```

## Usage

```bash
ai "explain how goroutines work"
```

## Requirements

- Go 1.21+
- API key (set in environment)
